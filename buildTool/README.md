# ДЗ "Настройка сборки проекта"
Проект с домашним заданием по настройки сборки, я использовала сборщик Vite

# Установка и локальный запуск

# Клонировать репозиторий
git clone https://github.com/EvaGoldynskaya/homework/tree/main
cd buildTool

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

## Другие команды

# Сборка проекта
npm run build

# Запуск предварительно собранного проекта из ./dist
npm run preview

# Настройка инструментов при установке зависимости
npm run prepare
