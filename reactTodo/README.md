Проект с домашним заданием по созданию приложения на React и использования кастомных хуков

# Команды

Запуск в режиме разработки

```
npm run dev
```

Сборка проекта

```
npm run build
```

Запуск предварительно собранного проекта из ./dist

```
npm run preview
```

Запуск тестов

```
npm run test
```

Настройка инструментов при установке зависимости

```
npm run prepare
```

Локальная проверка и исправление кода линтером

```
npm run lint
```

Локальное форматирование кода Prettier

```
npm run format
```

CI - проверка кода линтером (ошибка если есть проблемы)

```
npm run lint:check
```

CI -проверка форматирования кода (ошибка если есть проблемы)
`npm run format:check`

CI - все вышеуказанные проверки (CI) одной командой для удобства

```
npm run check-all
```
